# Headache analyser


Takes a google calendar export with predifned format and analyses it using python.


Google calendar headache entry template: 
```
Start: 0900
Times: 
  1200: 3/10
  1500: 4/10
  2050: 7/10
Coffe: no
Wine: no 
Sleep offset: 0 
Sleep deprivation: 0
Maxalt: no 
Maxalt fixed it:
```

Tip: add calendar entries to a separate calendar than your main one.
